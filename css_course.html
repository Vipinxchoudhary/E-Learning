CSS Basics

CSS (Cascading Style Sheets) is used to style and lay out web pages by specifying how elements should look.

body {
    background-color: lightblue;
}
h1 {
    color: navy;
    text-align: center;
}

CSS Selectors

CSS selectors allow you to target HTML elements to apply styles. Popular selectors include element, class, and ID selectors.

/* Element Selector */
p {
    color: green;
}

/* Class Selector */
.myClass {
    font-size: 18px;
}

/* ID Selector */
#myID {
    background-color: yellow;
}

CSS Box Model

The CSS box model describes the layout of HTML elements, including padding, borders, and margins.

div {
    border: 1px solid black;
    padding: 10px;
    margin: 20px;
}
