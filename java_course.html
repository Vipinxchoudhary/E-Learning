Java Basics

Java is a high-level, object-oriented programming language that is platform-independent and widely used for building applications.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Data Types

Java supports a variety of data types, such as integers, floats, strings, and booleans.

int myInt = 10;
double myDouble = 10.5;
String myString = "Hello, Java!";
boolean isJavaFun = true;

Java Methods

Methods in Java are blocks of code that perform specific tasks and can be reused throughout the program.

public class MyClass {
    public static void main(String[] args) {
        sayHello();
    }

    public static void sayHello() {
        System.out.println("Hello, Java!");
    }
}
