HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It structures the content on your website with elements like paragraphs, headings, links, and more.

<h1>This is a Heading</h1>
<p>This is a paragraph</p>

CSS Basics

CSS (Cascading Style Sheets) is used to control the presentation of your web page. It allows you to add colors, layout, fonts, and spacing.

body {
    background-color: lightblue;
}
h1 {
    color: navy;
}

HTML Example

Below is a simple example of an HTML page:

<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>Welcome to my website</h1>
    <p>This is a paragraph</p>
  </body>
</html>

CSS Example

Here’s an example of how to style your HTML with CSS:

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}
h1 {
    color: #4CAF50;
}
